# A-whole-body-atlas-of-enhancer-and-super-enhancer-regulatory-networks-in-pigs
Code of analysis during study "A whole-body atlas of enhancer and super-enhancer regulatory networks reveals tissue-specific hub enhancers and hierarchical regulatory logic in pigs"

## 1.Introduction

Genome-wide profiling of enhancer-centered regulatory architectures in pigs and establishes a hierarchical framework linking typical enhancers, super-enhancers, hub enhancers, and single-cell regulatory activity. 

## 2.Main content
Construction of enhacner regulatory network.
